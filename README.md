# bash-cli-python-django-angularjs-setup

## Description
Creates a simple django project
with angularjs mixed in.

## Tech stack
- bash

## Docker stack
- ubuntu:latest

## To run
`sudo ./install.sh -u`
http://localhost:8000

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
